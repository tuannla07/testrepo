# Matomo OmniaSSO Plugin

## Description

Will single sign on the user from Omnia.
Username needs to be equal in both Matomo and in Omnia

## Installation

In the config.ini.php file a section should be added.

``` 
[OmniaSSO]
clientId = "GUID"
clientSecret = "SECRET"
omniaUrl = "URL"
```
